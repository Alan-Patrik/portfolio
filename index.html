Alan Patrik

Um Desenvolvedor Full Stack.

Sempre nutri interesse e curiosidade sobre o funcionamento dos sistemas, levando-me, em 2020, a adentrar o curso de Residência de Software, onde tive meu primeiro contato com programação. Este marcou o início de minha jornada como desenvolvedor. O curso, de intensidade marcante, abordou temas até então desconhecidos, essenciais para a formação de um Full Stack. Após sua conclusão, dediquei-me continuamente ao aprimoramento, participando, nos anos seguintes, de um bootcamp Web Full Stack em 2021 e 2022. Essa experiência proporcionou-me conhecimentos avançados, tornando-me um profissional mais completo e preparado.

Sobre Mim

Sou casado, tenho 27 anos e moro em Petrópolis no estado do Rio de Janeiro. Gosto de música, toco instrumentos como bateria e saxofone. Faço backing vocal em shows aos finais de semana e amo praticar esportes.

Trabalho com tecnologia há mais de 3 anos, por qual sou apaixonado. Mergulho de cabeça em cada projeto com entusiasmo e habilidade técnica. Desde a arquitetura do sistema até a interface do usuário, busco constantemente maneiras de aprimorar e otimizar cada aspecto. Minha paixão pela tecnologia transcende a profissão, impulsionando-me a aprender e evoluir continuamente. Não há nada mais gratificante do que ver uma ideia ganhar vida por meio do meu trabalho, contribuindo para moldar o mundo digital e criar experiências significativas para os usuários. Ser um desenvolvedor full stack é mais do que escrever código; é criar o futuro, um algoritmo de cada vez.

Formação Educacional

Curso Análise e Desenvolvimento de Sistemas (Graduação)

Julho de 2023 até o atual momento @Uninter

  • O curso de Análise e Desenvolvimento de Sistemas na Uninter vem proporcionando uma formação abrangente e multidisciplinar, capacitando-me com habilidades essenciais para atuar no mercado de tecnologia da informação. Através do estudo de temas como Banco de Dados Relacional e Fundamentos da Programação Web, estou sendo preparado para projetar e desenvolver sistemas eficientes e interativos, fundamentais para atender às demandas atuais do mercado. Além disso, o curso enfatiza a importância da ética e responsabilidade socioambiental, incentivando a considerarem o impacto social e ambiental de nossas atividades profissionais.

Os fundamentos de sistemas de informação e desenvolvimento de software forneceram uma base sólida para o entendimento do ciclo de vida do desenvolvimento de sistemas e das práticas recomendadas para o desenvolvimento de soluções de software eficazes e sustentáveis. Com foco em lógica de programação e algoritmos, venho adquirindo habilidades fundamentais para a resolução de problemas e a implementação de soluções eficientes em sistemas de informação.

Web Full-Stack | Let's Code - SP

Agosto de 2021 – Junho de 2022 @santander-coders

  • O curso Web Full-Stack da Let's Code ofereceu uma jornada abrangente pelo desenvolvimento de aplicações web, abordando desde os fundamentos até tópicos avançados. Nos primeiros módulos, aprendi os conceitos básicos de HTML, CSS e lógica de programação, essenciais para construir interfaces de usuário estáticas e entender os princípios de programação. Em seguida, o curso avança para a programação orientada a objetos em Java, fornecendo uma base sólida para a criação de estruturas de código robustas e escaláveis.
  • Com o desenvolvimento do front-end, conheci o framework Angular, explorando componentes, forms e integração com APIs HTTP. Paralelamente, vi o JavaScript para interações dinâmicas no navegador e manipulação do DOM. Nos módulos posteriores, o foco se voltou para o back-end, onde foram explorados temas como bancos de dados, framework Spring, e tópicos avançados como reflexão em Java e programação reativa. Por fim, aprendi o deployment em contêineres Docker e nuvens como Heroku, além de explorar arquiteturas de microsserviços para projetos escaláveis e resilientes.

Programação orientada a Objetos

Março de 2021 – Julho de 2021 @Udemy

  • Durante o curso completo de Java em Programação Orientada a Objetos, adquiri uma compreensão abrangente dos princípios fundamentais da POO, desde classes e objetos até herança e polimorfismo. Avançando dos conceitos básicos para temas mais complexos, como interfaces e classes abstratas, desenvolvo habilidades sólidas para criar e manipular objetos de forma eficiente.

Utilizando IDEs como Eclipse e IntelliJ IDEA, e ferramentas de gerenciamento de dependências como Maven e Gradle, desenvolvo projetos que abrangem desde a lógica de negócios até a interface do usuário.
  • Por meio de projetos práticos desafiadores, consolidei meu conhecimento e desenvolvi habilidades práticas aplicáveis em ambientes profissionais.

Curso de Inglês Open English

Agosto de 2021 até o atual momento @open english

  • No primeiro ano, concentrei-me em estabelecer uma base sólida, focando principalmente na gramática, vocabulário básico e habilidades de conversação simples. Participar de aulas regulares e praticar com colegas foram partes essenciais do processo de aprendizagem.
  • No segundo ano, avancei para um nível intermediário, explorando tópicos mais complexos, como verbos modais, tempos verbais adicionais e estruturas de sentença mais elaboradas. Além disso, dediquei mais tempo à compreensão auditiva, assistindo a filmes e séries em inglês. Isso me ajudou a melhorar minha capacidade de entender o idioma quando falado com sotaques diferentes.
  • Neste terceiro ano, focarei em aprimorar minhas habilidades de comunicação oral e escrita. Além disso, intensifir meu estudo no vocabulário especializado em áreas de interesse pessoal e profissional.

Técnico de Programador Web e Mobile

Junho de 2020 – Novembro de 2020 @serratec

  • Na Introdução ao desenvolvimento web, compreendi os princípios fundamentais da web, incluindo o papel do navegador, servidores e protocolos como HTTP.
  • No Front-end, dominei a estruturação e estilização de páginas com HTML/CSS, assim como os fundamentos do JavaScript para criar interatividade. Também explorei o uso do React.js para simplificar o desenvolvimento de interfaces complexas.
  • No Back-end, mergulhei nos conceitos de linguagens como JAVA, bancos de dados relacionais e integração com o back-end. Configurar servidores e criar APIs RESTful foram habilidades fundamentais que desenvolvi.
  • Na abordagem Fullstack, aprendi a integrar o front-end e back-end, utilizando requisições HTTP e o framework Spring Boot para acelerar o desenvolvimento. Conheci também a ferramenta Git para fazer o controle de versão. Além disso, Desenvolvi projetos de tamanho variado para aplicar os conceitos aprendidos ao longo do curso. E para o projeto final, participei de um HACKATHON onde tive que desenvolver em grupo, um MVP de um software.
